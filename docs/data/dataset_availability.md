## Regarding tracks

### Dataset availability (no hosting)

This repository **does not distribute** the **<ins>track</ins>** dataset used in this research.

### What our tools does (and does not do)

The tools **only transforms data you already have locally**. It **does not** automatically collect, crawl, or scrape content from the web.  
To reproduce our research inputs, you must collect mario kart wii tracks in `.szs` format. For example, you can use the [Tockdom](https://wiki.tockdom.com/wiki/Main_Page) website, including, in particular, pages [List of Custom Tracks](https://wiki.tockdom.com/wiki/List_of_Custom_Tracks) and [Custom Track Distributions](https://wiki.tockdom.com/wiki/Custom_Track_Distribution) (a distribution being a collection of tracks). A list of the tracks used will be released as the project progresses.
